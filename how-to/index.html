Download the Modpack

Get started by downloading the modpack ZIP file:

Download Now

After importing to your launcher, download the Simple Voice Mod .jar file and place it in the /mods/ folder using your launcher. ( You won't be able to use Proximity Voice otherwise. )

Prism Launcher

  1. Open Prism Launcher
  2. Click Add Instance > Import
  3. Select the downloaded modpack.zip
  4. Click OK and launch!

CurseForge

  1. Open the CurseForge app
  2. Go to Minecraft > Create Custom Profile
  3. Click Import and choose modpack.zip
  4. Hit Play to start!

Connect to the Server

  1. Open your Minecraft launcher and start the modpack.
  2. Go to Multiplayer > Add Server.
  3. Enter the server IP: mc.brap.fun
  4. Click Done, then join the server!
